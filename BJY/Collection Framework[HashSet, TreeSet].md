## Collection Framework[HashSet, TreeSet]

<br/>

### <u>Set 컬렉션</u>

* 순서가 존재하진 않는 비선형적인 자료구조를 갖는다.
* 저장 순서가 유지되지 않는다.
* 중복을 허용하지 않는다.
  * 객체를 중복해서 저장할 수 없어서 중복 데이터 제거 수단으로 사용된다.
* 하나의 null만 저장할 수 있음
